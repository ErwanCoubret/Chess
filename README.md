# ♟️👑 Chess Game

## Description

This is a simple chess game written in C++ for a Engineering course at Télécom Physique Strasbourg.

Made by Erwan Coubret

## Commands

### Build & Run
```
make run
```

### Level tests

Individual tests (number from 1 to 4 for the different levels)
```
make test_<number>
```

All tests
```
make tests
```

### Clean
```
make clean
```
